# EXPERIMENT 005

## Title
FastAPI Backend Integration for DeepFakeGuard

## Objective
Expose DeepFakeGuard as a REST API capable of processing images and videos.

---

## Implemented Features

### Image Endpoint
POST /predict/image

Capabilities:
- Upload image
- Face detection
- Deepfake prediction
- Grad-CAM heatmap generation

---

### Video Endpoint
POST /predict/video

Capabilities:
- Upload video
- Frame extraction
- Face detection
- Deepfake prediction
- Most suspicious frame detection
- Grad-CAM evidence generation

---

## API Documentation

Swagger UI available at:

http://127.0.0.1:8000/docs

---

## Results

Image Endpoint:
Successfully processed uploaded images and returned predictions.

Video Endpoint:
Successfully processed uploaded videos and returned:
- prediction
